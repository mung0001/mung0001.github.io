    Functions

    • {;}, ()를 이용해 명령 그룹을 만들게 되면 같은 context에서 실행이 된다. 이것은 명령 그룹전체가 하나의 명령처럼 실행되는 것과 같은 효과가 있다.

    • 앞에 함수명을 붙여서 함수정의를 하게 되면 일반 명령과 동일하게 사용될 수 있다.

    • {;}은 현재 shell에서 ()는 subshell에서 실행되므로 보통 함수를 정의할 때 {;}을 사용하지만 필요하다면 ()을 사용할 수도 있다.

    $ echo hello world | read var; echo "$var"
# 파이프로 인해 subshell에서 실행되어 "$var"는 값이 표시되지 않는다.

$ echo hello world | { read var; echo "$var"; }
# {}을 이용해 명령 그룹을 생성하면 read, echo 명령이 같은 context에서 실행되어 정상적으로 값이 표시된다.

$ echo hello; echo world > outfile
# hello는 터미널에 표시되고 world만 outfile에 저장된다.

$ { echo hello; echo world; } > outfile
# hello world가 모두 outfile에 저장된다.

$ time sleep 2 && sleep 3
# 첫번째 명령만 시각 측정이 된다.

$ time { sleep 2 && sleep 3; }
$ {}안에 있는 명령이 모두 측정된다.

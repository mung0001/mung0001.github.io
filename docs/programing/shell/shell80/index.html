    awk

    • AWK명령어는 Aho + Weinberger + Kernighan이 조합된 이름으로, awk는 파일로부터 레코드(record)를 선택하고, 선택된 레코드에 포함된 값을 조작하거나 데이터화하는 것을 목적으로 하는 프로그램을 의미한다.

    • awk 명령의 입력으로 지정된 파일로부터 데이터를 분류한 다음, 분류된 텍스트 데이터를 바탕으로 패턴 매칭 여부를 검사하거나 데이터 조작 및연산 등의 액션을 수행하고, 그 결과를 출력하는 기능을 수행한다.

    • awk는 awk programming language라는 프로그래밍 언어로 작성된 프로그램을 실행하며, 리눅스에서 쉘 스크립트로 작성된 파일이 리눅스 쉘에 실행되는 것을 떠올리면 awk가 awk programming language 문법으로 작성된 코드를 실행한다는 이 동일한 맥락이라 볼 수 있다.

    • awk는 기본적으로 입력 데이터를 라인(line) 단위의 레코드(record)로 인식하며, 각 레코드에 들어 있는 텍스트는 공백 문자(space, tab)으로 구분된 필드들로 분류되며, 이렇게 식별된 레코드 및 필드의 값들은 awk 프로그램에 의해 패턴 매칭 및 다양한 액션의 파라미터로 사용된다.

     

    awk 명령어 옵션

    • awk 명령의 기본 형식은 하단과 같다.
    $ awk [OPTION] [awk program] [ARGUMENT]
OPTION
-F          : 필드 구분 문자 지정
-f          : awk program 파일 경로 지정
-v          : awk program에서 사용될 특정 variable 값 지정

awk program
-f          : 옵션이 사용되지 않은 경우, awk가 실행한 awk program 코드 지정
ARGUMENT    : 입력 파일 지정 또는 variable 값 지정

     

    awk program

    • awk 명령의 기본 형식은 보기와 같이 사용가능한 옵션이 많지는 않지만, 대신 awk 사용자는 옵션의 사용보다는 awk program의 작성에 많은 수고와 노고를 들어 스크립트 형식의 프로그램이 언어로 작성되기 때문에, 작성 방법이 다양하다.
    pattrn { action }
# 기본적인 awk program의 구조

awk [OPTION] 'pattern { action }' [ARGUMENT]
# 기본적인 awk 명령의 구조

$ awk '{ print }' ./file.txt
# file.txt의 모든 레코드를 출력

$ awk '/p/' ./file.txt
# file.txt에서 p를 포함하는 레코드를 출력

    • pattrn과 action은 모두 생략이 가능하며, pattern을 생략하는 경우는 모든 레코드가 적용되고, action을 생략하면 print가 적용된다.

    • 또한 pattem이 생략되는 경우, 매칭 여부를 검사할 문자열 패턴 정보가 없기 때문에 모든 레코드가 선택되고, action을 생략하면, 기본 액션인 print가 실행되는 것을 의미한다.

    • pattern과 action에 작성되는 awk program코드엔느 다양한 표현식, 변수, 함수 등이 사용되며, 이 중 가장 중요한 변수를 레코드와 필드를 나태내는 변수로, 하나의 레코드는 $0, 레코드에 포함된 각 필드는 그 순서대로 $1, $2, …, $n으로 지칭된다.

    $ awk 'length($0) > 10 { print $3, $4, $5 }' ./file.txt
# 레코드 길이가 10 이상인 경우 세 번째, 네 번째, 다섯 번째 필드를 출력

$ awk 'BEGIN { print "TITLE : Field value 1,2" } {print $1, $2} END {print "Finished"}' file.txt
# BEGIN과 END는 특별 패턴으로 awk가 BEGIN 패턴을 식별하면 입력 데이터로부터 첫 번째 레코드를 처리하기 전까지 "BEGIN"에 지정된 액션을 실행하고, END패턴은 BEGIN과는 반대로 모든 레코드를 처리한 다음 END에 지정된 액션을 실행

     

    awk 명령 사용 예제

    $ awk [OPTION] [awk program] [ARGUMENT]
    명령어 옵션 설명
    awk ‘{ print }’ File 파일의 전체 내용을 출력
    awk ‘{ print $1 } File 필드 값을 출력
    awk ‘{ print “STR"$1, “STR"$2 }’ File 필드 값에 임의 문자열을 함께 출력
    awk ‘/STR/’ File 지정된 문자열을 포함하는 레코드만 출력
    awk ‘$1 == 10 { print $2 }’ File 특정 필드 값 비교를 통한 레코드 출력
    awk ‘{sum += $3} END { print sum }’ file 특정 필드들의 합 출력
    awk ‘{ for (i=2; i<=NF; i++) total += $i }; END { print “TOTAL :” total } file 여러 필드의 합 출력
    awk ‘{ sum = 0 } { sum += ($3+$4+$5) } { print $0, sum, sum/3 }’ file 레코드 단위로 필드 합 및 평균 값 출력
    awk ‘{print $1, $2, $3+2, $4, $5}’ File 필드에 연산을 수행한 결과 출력
    awk ' length($0) > 20’ File 레코드 또는 필드의 문자열 길이를 검사
    awk -f [AWK File] File 파일에 저장된 awk program 실행
    awk -F ‘:’ ‘{ print $1 }’ File 필드 구분 문자열을 변경
    awk ‘{ print $0 }’ File awk 실행 결과 레코드 정렬
    awk ‘NR == 2 { print $0; exit }’ File 특정 레코드만 출력
    awk ‘{ printf “%-3s %-8s %-4s %-4s %-4s\n”, $1, $2, $3, $4, $5}’ File 출력 필드 너비 지정
    awk ‘{max =0; for (i=3; i<NF; i++) max = ($i > max) ? $i : max ; print max}’ File 필드 중 최대 값 출력

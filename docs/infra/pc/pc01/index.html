    PC 운영체제

    Operationg System

    • 운영체제의 정의

      • 시스템의 각종 하드웨어 자원과 소프트웨어의 자원을 효과적으로 관리 운영하고, 사용자가 시스템을 편리하게 이용할 수 있도록 하는 시스템 소프트웨어를 의미

     

    • 운영체제의 목적

      • 1. 처리능력향상

      • 2. 응답시간 단축

      • 3. 사용 기능도 증대

      • 4. 신뢰도 향상

     

    • 운영체제의 종류

      • 서버용 : Unix, Linux Windows Server
      • 개인용 : DOS, windows 98, windows xp, OS2
      • 모바일 : Windows CE, los, 안드로이드(proyo, gingerbread..), 바다

    Mail Server

    E-Mail 작동 개념

    • SMTP(Simple Mail Transfer Protocol)을 사용하여 클라이언트가 메일을 보내거나, 메일 서버끼리 메일을 주고 받습니다.

    • POP3(Post Office Protocol)을 사용하여 메일 서버에 도착되어 있는 메일을 클라이언트로 가져옵니다.

    • IMAP(Internet Mail Access Protocol)은 POP3와 동일합니다.

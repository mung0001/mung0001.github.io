    Windows CLI #

    #

    • Windows Server는 GUI환경이 아닌 도스환경으로도 실행될 수 있으며, GUI 환경이 아닌 만큼 리소스의 낭비가 적은 특징을 가지고 있습니다.

$ echo %computername%

# 컴퓨터의 이름을 CORE로 변경

# 0/ 초/ 안에 reboot를 진행

netsh> interface ipv4 show interface
# 인터페이스 관련 정보를 출력

netsh> interface ipv4 set address name="3" source=static address=192.168.10.50 mask=255.255.255.0 gateway=192.168.10.2
# idx의 값이 3인 인터페이스의 ipv4의 주소를 192.168.10.50, 넷마스크 255.255.255.0 게이트웨이를 192.168.10.2로 고정

netsh> interface ipv4 add dnsserver name="3" address=8.8.8.8 index=1
# idx의 값이 3인 인터페이스의 dns 주소를 8.8.8.8으로 설정

    Window PowerShell #

    PowerShell이란? #

    #

    • Unix의 셸과 같은 기능을 제공

    • 윈도우 서버 2008에서 처음 소개

    • Windows Server를 관리할 때 자주 사용되는 것들

    • PowerShell에서 스크립트로 만들어 놓으면 언제든지 일관된 재사용이 가능

    #

     

    #

    PowerShell의 특징 #

    #

    • PowerShell은 PowerShell 콘솔에서 대화형으로 사용

    • PowerShell은 텍스트로 구성, 메모장에서 편집가능

    • PowerShell은 대개 대소문자를 구분하지 않음

    • 명령어나 경로의 Tab 자동 완성 기능을 제공

    • 기존 DOS 명령을 대부분 사용 가능

    #

     

    #

    • PowerShell 진입
    $ powershell
# powershell로 쉘을 변경합니다.

    #

     

    #

    • 설치되어 있는 패키지 확인
    PS $ Get-WindowsFeature

    #

     

    #

    • 설치
    PS $ Install-WindowsFeature DHCP -Source wim:D:\Sources\install.wim:1
# DHCP 서버 설치

    #

     

    #

    • 문자 출력
    PS $ write-output "문자"

    #

     

    #

    • PC 정보출력
    PS $ Get-Host

    #

     

    #

    • 파워셸 업데이트
    PS $ Update-help

    #

     

    #

    • 도움말 출력
    PS $ Get-help 명령어
#

    #

     

    #

    • 파일에 기록
    PS $ Add-Content "테스트" -path C:\myfile.txt
# 테스트라는 내용을 C:\myfile.txt에 기록한다.

    #

     

    #

    • 파일을 읽음
    PS $ Get-Content 파일경로

    #

     

    #

    • 파일을 복사
    PS $ Copy-item 복사할 파일 보사할 위치

    #

     

    #

    • 파일을 나열
    PS $ Get-ChildItem 경로

    #

     

    #

    • 명령어 출력
    PS $ Get-Command 명령어
# Get-Command Write

    #

     

    #

    • 파워셸 ISE 실행
    PS $ PowerShell_ISE
#

    #

     

    #

    • 현재 프로세스 출력
    PS $ Get-Process

    #

     

    #

    • 결과값을 저장
    PS $ 명령어 | Export-CSV 경로

    #

     

    #

    • 노트패드로 실행
    $ notepad 경로

    #

     

    #

    • 보기 편한 모양으로 출력
    $ 명렁어 | Out-GridView

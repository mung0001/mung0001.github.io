    SSL 보안 인증서 발급받기

    • 이미 도메인 주소가 있으신 분은 그것을 사용하고, 없으신 분은 Link에서 무료 도메인을 사용하시면 됩니다.

    VMware에 NAS 설치

    • Vmware에서 Custom으로 Virtual Machine을 설치합니다.

     


    • 14 혹은 15 버전을 선택 후 iso파일을 선택합니다.
    • 파일이 없을시 Link에서 다운로드 합니다..

     

    • 운영체제는 Linux에서 2.6.x Kernel 64-bit를 선택합니다.

     

    • Virtual Machine의 이름을 지정합니다.

     

    • 코어 할당량을 지정합니다. ( 여기서는 빠른 설치를 위해 2/2개를 할당하였습니다. )

     

    • RAM을 할당을 지정합니다. ( 여기서는 빠른 설치를 위해 4G을 할당하였습니다.

     

    • 접속의 편리를 위해 Nat로 선택합니다.

     

    • LSILogic으르 선택합니다.

     

    • SATA를 선택합니다.



    • Use an existing virtual disk를 선택 후, synoboot.vmdk를 임포트 시킵니다.

     

    • 설정 창이 뜨면 기존 포맷을 유지시킵니다.

     




    • NAS를 실행시키기 전에 하드를 추가시킵니다. ( 여기서는 20G 하나만 추가시키겠습니다. )

     

    • 실행 후, 1분이 경과하면 find.synology.com에 접속합니다.

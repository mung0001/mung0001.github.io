    • 실습 프로그램 - ICQA Router Emulator

     

    Router IP/ Subnet 설정

    1. en(enable) 명령어 입력
      • 사용자 모드에서 관라자 모드로 전환
    Router> en
Router#

    1. conf t(configure terminal) 명령어 입력 r - 관리자 모드에서 전력 설정 모드로 전환
    Router# conf t
Router(config)#

    1. int f0/0(interface fastethernet 0/0) 명령어 입력
      • 제시된 인터페이스로 이동
    Router(config)# int f0/0
Router(config-if)#

    1. IP/ Subnet 설정 명령어 입력
      • ip addr IPaddress subnetmask
    Router(config-if)# ip addr 192.168.0.10 255.255.255.0
Router(config-if)# no sh (no shutdown)

    1. copy r s관리자 모드에서 저장
      • exit명령어로 나온 후 저장
    Router(config-if)# exit
Router(config)# exit
Router# copy r s

     

    대역폭 설정

    1. en(enable) 명령어 입력
      • 사용자 모드에서 관라자 모드로 전환
    Router> en
Router#

    1. conf t(configure terminal) 명령어 입력
      • 관리자 모드에서 전력 설정 모드로 전환
    Router# conf t
Router(config)#

    1. int f0/0(interface fastethernet 0/0) 명령어 입력
      • 제시된 인터페이스로 이동
    Router(config)# int f0/0
Router(config-if)#

    1. bandwidth 2048 명령어 입력
      • 명령어를 입력하여 대역폭을 설정
    Router(config-if)# bandwidth 2048

    1. clock rate 72000 명령어 입력
      • 명령어를 입력하여 클럭 속도를 설정(K의 경우 1000단위)
    Router(config-if)# clock rate 72000
or
Router(config-if)# clock rate 72K

    1. copy r s관리자 모드에서 저장
      • exit명령어로 나온 후 저장
    Router(config-if)# exit
Router(config)# exit
Router# copy r s

    기본 게이트웨이 설정

      • 사용자 모드에서 관라자 모드로 전환
    Router> en
Router#

    1. conf t(configure terminal) 명령어 입력
      • 관리자 모드에서 전력 설정 모드로 전환
    Router# conf t

    1. ip default-gateway 192.168.0.1 명령어 입력
      • 명령어를 입력하여 기본 게이트웨이를 지정

    1. ip dhcp pool icqa / network 명령어 입력
      • 명령어를 입력하여 DHCP 서버 이름과 주소를 설정
    Router(config)# ip dhcp pool icqa
Router(dhcp-config)# network 192.168.10.0 255.255.255.0

      • 명령어를 입력하여 Telnet의 접속이 가능
    Router(config-line)# login

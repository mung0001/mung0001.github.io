    Router 명령어 / 3 * 5.5점

    • 실습 프로그램 - ICQA Router Emulator

     

    설정 명령어

    명령어 설명
    enable en 관리자 모드로 전환 –> Router#
    end 관리자 모드로 전환
    configure terminal conf t 관리자 모드에서 설정모드로 전환 –> Router(config)#
    hostname 이름 호스트 이름 지정
    login 로그인 명령
    exit 이전모드로 복귀
    line console 0 li c 0 콘솔라인 모드 전환 명령
    line vty li v 0 4 텔넷 등 가상 터미널 전환 명령
    enable password 암호명 관리자모드로 엑세스할 때 암호 설정 / (암호화되지 않고 저장)
    enable secret 암호명 관리자모드로 엑세스할 때 암호 설정 / (암호화되어 저장)
    erase startup-config 라우터 설정 명령을 지우고 초기화
    clock set 시간 장비의 시간을 설정할 때 사용
    exec-timeout 시간 exec-t 시간 콘솔 종료대기 시간변경 / (exec-timeout 0 0: 종료시간 없음)
    copy r s 라우터 셋팅 내용을 NVRAM에 저장

     

    확인 명령어

    명렁어 설명
    show run 설정된 인터페이스 확인
    show clock 설정된 시간을 확인
    show flash 플래쉬 확인
    show config 라우터 장비설정의 저장내용을 확인하는 명령
    show process 프로세스 정보확인
    show history 사용했던 전체 명령어 확인
    show version IOS의 버전정보 확인과 S/W, H/W정보 확인
    show ip route 라우팅 테이블 확인
    show ip interface brief 인터페이스 정보 확인
    show startup-config NVRAM NVRAM 정보 확인

    Router IP/ Subnet 설정

    1. en(enable) 명령어 입력
      • 사용자 모드에서 관라자 모드로 전환
    Router> en
Router#

    1. conf t(configure terminal) 명령어 입력 r - 관리자 모드에서 전력 설정 모드로 전환
    Router# conf t
Router(config)#

    1. int f0/0(interface fastethernet 0/0) 명령어 입력
      • 제시된 인터페이스로 이동
    Router(config)# int f0/0
Router(config-if)#

    1. IP/ Subnet 설정 명령어 입력
      • ip addr IPaddress subnetmask
    Router(config-if)# ip addr 192.168.0.10 255.255.255.0
Router(config-if)# no sh (no shutdown)

Router(config-if)# ip directed-broadcast
# broadcast 설정 ( 선택 )

    1. copy r s관리자 모드에서 저장
      • exit명령어로 나온 후 저장
    Router(config-if)# exit


Router(config)# ip domain-name DOMAIN
# DOMAIN 설정 ( 선택 )

Router(config)# exit
Router# copy r s

    Linux 명령어 / 2 * 5.5점

    단답형

    • 윈도우의 ipconfig 명령어와 같이 리눅스에서 ip address 정보를 확인하는 명령어는?
    $ ifconfig

     

    • 리눅스에서 파일이나 폴더를 찾을 때 사용하는 명령어는?
    $ find

     

    • 리눅스에서 현재 설치된 하드디스크의 용량을 확인하는 명령어는?
    $ df

     

    • 리눅스에서 현재 위치를 확인하고 싶을 때 사용하는 명령어는?
    $ pwd

     

    • 리눅스에서 포트/프로토콜 정보를 확인할 수 있는 명령어는?
    $ /etc/services 또는 cat /etc/services

     

    • 리눅스에서 CPU 메모리 사용 정보를 확인하는 명령어는?
    $ top

     

    • 리눅스에서 도움말-메뉴얼을 확인하는 명령어는?
    $ man

     

    • 리눅스에서 파일 또는 디렉토리를 삭제하는 명령어는?
    $ rm

     

    • 리눅스에서 네트워크 연결 상태, 라우팅 테이블 정보 등을 보여주는 명령어는?
    $ netstat

     

    • 리눅스에서 물리적 장치를 특정한 위치로 연결 시켜주는 과정을 마운트라고 한다. 마운트를 하는 명령어는?
    $ mount

     

    • 리눅스에서 물리적 장치를 특정한 위치로 연결 시켜주는 과정을 마운트라고 한다. 마운트를 해제하는 명령어는?
    $ umount

     

    • 리눅스에서 계정을 생성하는 과정이 사진으로 나왔는데, 그럴 경우 답은~
    $ passwd

     

    • 리눅스의 이동 명령어는?
    $ mv

     

    • 리눅스의 파일 타입 확인 명령어는?
    $ file

     

    • 리눅스의 복사 명령어는?
    $ cp

     

    • 리눅스에서 강제 종료를 하는 명령어는?
    $ reboot, shutdown -r now, init 0, halt

     

    • 리눅스에서 권한 설정 명령어는?
    $ chmod

     

    • 특정 호스트 또는 연결된 노드를 알아보는 리눅스 명령어는?
    $ traceroute

     

    • 호스트 컴퓨터가 TCP/IP 네트워크 및 네트워크 리소스에 연결 가능한지 확인할 수 있는 리눅스 명령어는?
    $ ping

     

    • 라우팅 테이블을 확인하거나 수정하는 리눅스 명령어는?
    $ route

     

    • 네임 서버에 도메인 이름이나 ip를 조회하는 리눅스 명령어는?
    $ nslookup

     

    • Telnet이나 콘솔로 로그인된 모든 사용자에게 메시지를 전송하는 리눅스 명령어는?
    $ wall

     

    • 현재 접속된 사용자와 프로세스 상태를 확인하는 리눅스 명령어는?
    $ who

     

    • 파일이 어떤 디렉토리에 있는지 표시하는 리눅스 명령어는?
    $ which

     

    • 파일을 생성하거나 파일의 액세스한 시간을 변경하는 리눅스 명령어는?
    $ touch

     

    • 특정 사용자의 정보를 출력하는 리눅스 명령어는?
    $ finger

     

    • 디렉터리별 디스크 사용 정보를 확인하는 명령어는?
    $ du

     

    • 시스템을 종료하는 리눅스 명령어는?
    $ halt
(halt -f : 시스템 강제 종료)

     

    • icqa 폴더 및 하위 폴더 subdir를 동시에 생성하는 리눅스 명령어는?
    $ mkdir -p icqa/subdir

     

    • icqa 폴더 및 하위 폴더 subdir를 동시에 삭제하는 리눅스 명령어는?
    $ rm -r icqa/subdir

     

    • HWP 프로그램 설치 후 어디에 설치되었는지 확인하는 리눅스 명령어는?
    $ whereis HWP

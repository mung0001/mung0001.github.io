    Djnago

    Djnago 설치

    설치환경

    • Ubuntu 18.04

    • mysql 5.7

    • python 3.6.8

    • pip3 9.0.3

    • django 2.1

    Ubuntu 설정

    • apt-get update -y

    • apt-get upgrade -y

    • apt-get install python3 -y

    • apt-get install python3-pip -y

    • apt-get install gcc -y

    • apt-get install python-dev -y

    • apt-get install libmysqlclient-dev -y

    • apt-get install python3-devel -y

    mysql 설치

    • wget https://dev.mysql.com/get/mysql57-community-release-el7-11.noarch.rpm

    • sudo rpm -ivh mysql57-community-release-el7-11.noarch.rpm

    • sudo apt-get install mysql-server

    • systemctl start mysql

    mysql 임시 비밀번호와 비밀번호 변경

    • 임시 비밀번호 입력 후 변경할 비밀번호를 입력

    • cat /var/log/mysqld.log | grep root@localhost ( 임시 비밀번호가 나오지 않을 경우 엔터 입력 )

    • mysql_secure_installation

    • mysql -u root -p 비밀번호 입력

    한글 입력 출력 설정

    1. mysql을 실행하여 아래와 같이 입력
    mysql> show variables like 'c%';

    • 몇몇 설정이 latin1으로 되어있는 것을 확인할 수있다.

    • 테이블을 수정하면 일시적으로 한글 입력을 확인할 수 있다.

    • 영구적으로 적용하기 위해 config파일을 수정할 필요가 있다.

    1. /etc/mysql/mysql.conf.d/ 에 다음과 같은 파일들을 추가한다.
    # client.cnf

[client]
default-character-set=utf8

    # mysqld.cnf 

[mysqld]
character-set-server=utf8
collation-server=utf8_general_ci
init_connect=SET collation_connection=utf8_general_ci
init_connect=SET NAMES utf8

    # mysqldump.cnf

[mysqldump]
default-character-set=utf8

    # mysql.cnf

[mysql]
default-character-set=utf8

    1. mysql 재시작 및 확인
    systemctl restart mysql
mysql -u root -p
mysql> show variables like 'c%';
mysql> status;

    개발환경 설정

    • virtualenv 환경
      • pip3 install –user virtualenv
      • source etc/profile
      • virtualenv ve

    • virtualenv 활성화 비활성화
      • source ve/bin/activate
      • (ve) deactivate

    • django 2.1 설치

      • (ve) pip3 install django==2.1

    • virtualenv 환경에 설치된 목록 확인

      • (ve) pip freeze

    • django에서 mysql을 사용하기위한 모듈 설치

      • (ve) pip install mysqlclient

    • 오류가 난다면 아래를 설치한 뒤 다시 시도한다.
      • yum install gcc
      • yum install python-devel mysql-devel
      • yum install python3-devel

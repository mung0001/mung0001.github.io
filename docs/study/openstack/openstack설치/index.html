    자동화 툴을 이용한 설치 2) 준비 가상머신 3대 cpu 메모리 디스크 controller 1 3gb 50gb compute 1 2gb 50gb network 1 1gb 20gb

    각각 맞는 대역을 설정

    • yum -y install bind-utils

     

    1. 전체 설치 과정
    • 호스트명 설정 controller | comupte | network 노드
    • vi /etc/hosts 아래에 각 노드의 내용을 추가
    192.168.0.62 controller
192.168.0.162 compute
192.168.0.163 network

    • controller | compute | network의 hostname에 기존 내용 지우고 다음과 같이 수정
    • vi/etc/hostname
    controller  192.168.0.62
compute	192.168.0.162
network	192.168.0.166

    • 재시작

      2. 방화벽 설정

    • controller 노드
    systemctl stop firewalld
systemctl disable firewalld
systemctl stop NetworkManager
systemctl disable NetworkManager
setenforce 0
sed -i 's/=enforcing/=disabled/g' /etc/sysconfig/selinux

    • compute 노드
    systemctl stop firewalld
systemctl disable firewalld
systemctl stop NetworkManager
systemctl disable NetworkManager
setenforce 0
sed -i 's/=enforcing/=disabled/g' /etc/sysconfig/selinux

    • network 노드
    systemctl stop firewalld
systemctl disable firewalld
systemctl stop NetworkManager
systemctl disable NetworkManager
systemctl status firewalld
setenforce 0
sed -i 's/=enforcing/=disabled/g' /etc/sysconfig/selinux
    • 전체 재시작

      4. 오픈스택 설치 파일 다운

    • controller 노드
    yum update -y
yum install centos-release-openstack-rocky -y
yum install openstack-packstack -y
yum install openstack-utils -y
yum install chrony ntp -y
    1. 오픈스택 설치 파일 설정
    • controller 노드
    packstack --gen-answer-file=/root/rocky-answer.txt


    vi /root/rocky-answer.txt
CONFIG_CONTROLLER_HOST=[컨트롤러 노드 IP]
# controller Node
CONFIG_COMPUTE_HOSTS=[컴퓨트 노드 IP],[컴퓨트 노드 IP],[컴퓨트 노드 IP]
# compute Node, 
CONFIG_NETWORK_HOSTS=[네트워크 노드 IP],[네트워크 노드 IP]
# Network node
CONFIG_PROVISION_DEMO=n
CONFIG_NTP_SERVERS=0.centos.pool.ntp.org iburst, 1.centos.pool.ntp.org iburst, 2.centos.pool.ntp.org iburst, 3.centos.pool.ntp.org iburst
CONFIG_CINDER_VOLUMES_SIZE=50G
    1. 오픈스택 설치 진행
    • controller 노드
    packstack --answer-file=/root/rocky-answer.txt
